# Program Platform Starter Server

Local-first NestJS API.

Features:
- File-backed starter data in `server/data/store.json`
- Session auth with seeded demo users
- Generic curriculum, community, deliverables, feedback, notifications, admin summary

Seeded accounts:
- `admin@example.com` / `Starter123!`
- `faculty@example.com` / `Starter123!`
- `learner@example.com` / `Starter123!`
